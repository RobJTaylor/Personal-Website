Espresso

Espresso is a small application that I made in the summer of 2019. During this time, I was archiving a large number of files, and required my computer to stay awake 24/7.

I could simply adjust the Windows power plan to achieve this, but that would be too easy. I set out to create a modern Windows application in the vein of Caffeine.

The result of this was Espresso - a modern Windows application using XAML Islands to provide a user interface built on the latest framework by Microsoft. It's a small application that lives in the notification area to provide an experience similar to caffeine.

I plan to add more features to Espresso in the future, when WinUI 3.0 is available, as this will remove the dependency on XAML Islands and make maintenance much easier. The current release is available from the Microsoft Store.
